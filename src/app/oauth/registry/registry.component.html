To access any FTC restful API, the request must present a valid OAuth access token. Do not mix the usage the app tokens with personal keys. Personal keys are meant to be used by humans to access the API endpoints manually.

App Token Usage

Set the Authorization HTTP request header:

Authorization: Bearer access_token
